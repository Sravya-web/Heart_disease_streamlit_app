# Heart Disease Prediction App (Streamlit)

A simple and interactive Streamlit web app that predicts the risk of heart disease using a Machine Learning model.  
Users can enter basic health and lifestyle details to get a quick prediction.

---

## Features
- User-friendly Streamlit UI  
- Takes health metrics as input (Age, BP, Heart Rate, Sleep, etc.)  
- ML-based prediction using Scikit-learn  
- Real-time prediction output  
- Beginner-friendly project structure  

---

## Tech Stack
- Python  
- Streamlit  
- Scikit-learn  
- Pandas  
- NumPy  

---

## Input Features Used
- Age  
- Gender  
- Height (cm)  
- Weight (kg)  
- Daily Steps  
- Calories Intake  
- Hours of Sleep  
- Heart Rate  
- Exercise Hours per Week  
- Smoker  
- Alcohol Consumption per Week  
- Diabetic  
- BP Systolic  
- BP Diastolic  

---

How to Run Locally

# Clone the repository
git clone https://github.com/<your-username>/heart_disease_project.git

# Go to project folder
cd heart_disease_project

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
